# SQL Data Portfolio

Welcome to my SQL Data Portfolio! This repository contains real-world data analysis projects demonstrating advanced SQL skills using MySQL.

Projects Included

 1. Sales Performance Analysis
Comprehensive sales analysis including revenue trends, product performance, and customer segments.
- **Skills**: Window Functions, CTEs, Advanced Aggregations
- **Location**: `project_1_sales_analysis/`

 2. Customer Churn Analysis
Predictive analysis to identify at-risk customers and retention strategies.
- **Skills**: Time-series Analysis, Segmentation, Statistical Queries
- **Location**: `project_2_customer_churn/`

 3. Inventory Management System
Real-time inventory tracking and optimization recommendations.
- **Skills**: JOINs, Inventory Calculations, Reorder Analysis
- **Location**: `project_3_inventory_management/`

Technologies Used
- **Database**: MySQL
- **Concepts**: Complex Joins, Window Functions, CTEs, Aggregations, Subqueries
- **Tools**: SQL queries, Data Analysis

Project Structure
Each project contains:
- `project_description.md` - Business context and objectives
- `schema.sql` - Database schema definition
- `insert_data.sql` - Sample data
- `analysis_queries.sql` - Analysis queries with results

Getting Started
1. Choose a project folder
2. Create the database schema using `schema.sql`    
3. Insert sample data using `insert_data.sql`
4. Run queries from `analysis_queries.sql`
